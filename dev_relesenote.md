# Release Notes - MemMem V0.1.3

**วันที่อัปเดต:** 11 กรกฎาคม 2026
**เวอร์ชัน:** V0.1.3 (Scrollable Practice Card & Layout Fixes)

## 🚀 ฟีเจอร์ใหม่และการปรับปรุง (New Features & Improvements)

- **ระบบทบทวนคำศัพท์ (Flashcard Practice Screen):**
  - **Scrollable Card Back & Premium Notes UI:** ปรับแก้การจัดวางผังหน้าหลังของการ์ด (Card Back) ให้เป็นแบบเลื่อนหน้าจอดิ่งลงได้ (ScrollView) เมื่อมีเนื้อหาโน้ตคำอธิบายเพิ่มเติมที่ยาวมาก ป้องกันปัญหาตัวอักษรคำแปลและตัวโน้ตแสดงทับซ้อนกัน พร้อมทั้งแต่งดีไซน์กล่องข้อความโน้ตใหม่ให้เป็นกรอบสวยงามแบบไฮไลท์ขอบซ้ายสีฟ้าและมีสีพื้นหลังเปลี่ยนตามธีมสว่าง/มืด

# Release Notes - MemMem V0.1.2

**วันที่อัปเดต:** 10 กรกฎาคม 2026
**เวอร์ชัน:** V0.1.2 (Search UI Sync & Windows Local Build Setup)

## 🚀 ฟีเจอร์ใหม่และการปรับปรุง (New Features & Improvements)

- **ระบบทบทวนคำศัพท์ (Flashcard Practice Screen):**
  - **Edge Navigation (Hot Zones):** ปรับปรุงระบบนำทางข้ามการ์ด โดยเปลี่ยนจากปุ่มด้านล่างมาเป็นแถบสัมผัสด้านข้างจอฝั่งซ้าย-ขวา เพื่อเพิ่มความสะดวกในการใช้งานแบบมือเดียว (One-handed use)
  - **Restart/Reset Session:** เพิ่มปุ่ม Reset (ไอคอนรีเฟรช) ในแถบ Header ด้านขวาเพื่อเริ่มทบทวนเซ็ตคำศัพท์ปัจจุบันใหม่ทั้งหมดได้ทันที (เริ่มต้นใหม่จากใบแรก ล้างสถานะการพลิกการ์ด และสุ่มลำดับใหม่หากกำลังใช้งานโหมดสุ่มคำ)
- **การนำทางและฟังก์ชันการค้นหา (Transitions & Searches):**
  - **Seamless Search Transition:** เมื่อผู้ใช้คลิกเลือกผลการค้นหาจากหน้าหลัก ระบบจะสลับหน้าจอเข้าไปยังกลุ่มคำศัพท์ที่เลือก และเปิดหน้าต่าง Modal แสดงรายละเอียดคำศัพท์ใบนั้นให้โดยอัตโนมัติทันที
  - **Auto-Clear Search Input:** เมื่อทำการย้ายหน้าจอหรือกดย้อนกลับมายังหน้าหลัก ช่องการค้นหาจะถูกล้างค่าออกอัตโนมัติ (Reset) ป้องกันปัญหารายการค้นหาค้าง
  - **Instant Searchability on Import:** แก้ไขบั๊กการซิงค์ข้อมูล โดยทำให้คำศัพท์ใหม่ที่ได้จากการ Import ไฟล์เข้ามาพร้อมให้ค้นหาในระบบ Global Search ได้ทันทีโดยไม่ต้องเปลี่ยนหน้าหรือปิดเปิดแอปใหม่
- **ระบบจัดทำตัวติดตั้งภายในเครื่อง (Local Build Config):**
  - **Dynamic Build Version & Custom Filename:** พัฒนาระบบให้แอนดรอยด์ดึงเวอร์ชันจาก `app.json` มาใช้ตั้งชื่อผลลัพธ์เป็น `MemMem-release-<version>.apk` โดยอัตโนมัติเมื่อสั่งคอมไพล์ในเครื่องคอมพิวเตอร์ของคุณ

# Release Notes - MemMem V0.1.1

**วันที่อัปเดต:** 9 กรกฎาคม 2026
**เวอร์ชัน:** V0.1.1 (Flashcard & UI Fixes)

ยินดีต้อนรับสู่แอปพลิเคชัน MemMem (แอปจดบันทึกและทบทวนคำศัพท์ภาษาญี่ปุ่นออฟไลน์ 100%) นี่คือฟีเจอร์หลักและการปรับปรุงที่ถูกเพิ่มเข้ามาในเวอร์ชันแรกนี้:

## 🚀 ฟีเจอร์ใหม่ (New Features)

- **ระบบทบทวนคำศัพท์ (Flashcard Practice):** เพิ่มหน้าระบบสุ่มคำศัพท์เพื่อทบทวน แตะเพื่อพลิกดูคำแปล (Flip card) และมีปุ่มนำทาง (Prev/Next)
- **ระบบคำศัพท์ออฟไลน์ (Offline-First):** เพิ่มและจัดการชุดคำศัพท์ภาษาญี่ปุ่นได้โดยไม่ต้องใช้อินเทอร์เน็ต ข้อมูลทั้งหมดถูกบันทึกไว้ในเครื่องอย่างปลอดภัย
- **หมวดหมู่ชุดคำศัพท์ (Collections):** สร้าง ลบ และแก้ไขชื่อชุดคำศัพท์ได้อย่างอิสระ
- **ระบบเพิ่มคำศัพท์ (Vocab Management):** สามารถเพิ่มคำศัพท์ คันจิ คำอ่าน (Hiragana/Katakana) ความหมาย (ภาษาไทย) และแหล่งที่มาได้
- **ระบบป้องกันคำซ้ำ (Duplicate Check):** เมื่อมีการเพิ่มคำศัพท์ที่ซ้ำกัน ระบบจะแจ้งเตือนให้ผู้ใช้ตัดสินใจว่าจะบันทึกทับหรือไม่
- **Import / Export เป็นไฟล์ JSON:**
  - นำเข้าชุดคำศัพท์จากภายนอก (มีระบบเช็คชื่อไฟล์ซ้ำ และบังคับเปลี่ยนชื่อก่อน Import)
  - ส่งออกชุดคำศัพท์เพื่อแชร์ให้เพื่อน โดยสามารถตั้งชื่อไฟล์อิสระก่อน Export ได้ (ระบบจะจัดการแปลงช่องว่างเป็น `_` ให้ถูกต้องตามหลักการตั้งชื่อไฟล์อัตโนมัติ)
- **ระบบเรียงลำดับ (Sorting):** เรียงลำดับชุดคำศัพท์หรือคำศัพท์จาก "ใหม่สุดไปเก่าสุด" และ "เก่าสุดไปใหม่สุด"
- **ค้นหาคำศัพท์ (Global & Local Search):** ระบบค้นหาคำศัพท์ทั้งหมดข้ามชุดคำศัพท์ และค้นหาภายในชุดแบบรวดเร็ว
- **หน้า About & Privacy Policy:** หน้าแสดงรายละเอียดนักพัฒนา และนโยบายความเป็นส่วนตัวเพื่อเตรียมพร้อมสำหรับการนำขึ้นสโตร์

## 💅 การปรับแต่งหน้าตาแอป (UI/UX)

- **แก้ปัญหาภาษาไทยขาดหาย (Thai Typography Fix):** ปรับปรุง `lineHeight` และระบบตัดขอบของ Android ให้แสดงผลสระ ไม้โท และการเว้นวรรคภาษาไทยได้สมบูรณ์ ไม่โดนตัดขอบซ้ายขวาบนล่าง
- **แก้ปัญหากรอบพิมพ์คำศัพท์ (Android Modal Keyboard Fix):** ปรับปรุงหน้าต่าง Modal แบบ Bottom-Sheet ให้ไม่โดนคีย์บอร์ดทับ และใช้เทคนิค "Infinite Skirt" (ต่อขอบด้านล่าง) เพื่อลบช่องโหว่ใสเวลากดปิดคีย์บอร์ดบน Android ได้อย่างสมบูรณ์ 100%
- **Theme Toggle:** เพิ่มเมนู "Settings" มุมขวาบน เพื่อปรับเปลี่ยนธีมหน้าจอเป็น System Default, Light Mode หรือ Dark Mode ได้ตามต้องการ
- **ปุ่มเรียงลำดับ (Sort Button):** ออกแบบเป็นปุ่มลูกศรขึ้น/ลง พร้อมเปลี่ยนสีไฮไลท์เพื่อให้ดูเข้าใจง่าย
- **ปัดเพื่อปิด (Swipe to Dismiss):** Modal ต่างๆ ถูกออกแบบให้สามารถใช้สไลด์นิ้วลงเพื่อปิดหน้าต่างได้แบบ Native UX

## 🔧 ประสิทธิภาพและความปลอดภัย (Performance & Security)

- **Sanitize Input:** อุดช่องโหว่การโจมตีแบบ Path Traversal ในระบบจัดการไฟล์ เพื่อไม่ให้ไฟล์ถูกเขียนทับมั่วซั่ว
- **Memoization (React.memo & useMemo):** ปรับปรุงโครงสร้างของ List และระบบค้นหาให้สามารถพิมพ์ค้นหาได้ทันทีโดยไม่เกิดอาการกระตุก (Lag) แม้จะมีคำศัพท์นับพันคำ
- **FlatList Optimization:** ปรับจูนการเรนเดอร์ลิสต์ให้ประหยัด RAM เครื่อง
